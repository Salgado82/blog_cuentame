# BLOG CUENTAME

Blog cuentame realizado con node, express and mysql

## Primeros pasos

Para iniciar el proyecto es necesario crear la base de datos con el dump que se encuentra dento de la carpeta database.

El proyecto cuenta con archivo makefile para tener ordenados los comandos necesarios.

Lo siguiente para ejecutar el proyecto es ejecutar:

```
make start-env-file
```

Y esto va a generar el archivo .env donde tendremos que colocar los accesos de la base local o donde hayamos ejecutado el dump de la base de datos.

Lo siguiente es ejecutar el proyecto:

```
make start
```
